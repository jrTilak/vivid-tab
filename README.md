### Install from Chrome Web Store

ğŸ”— [Vivid Tab on Chrome Web Store](https://chrome.google.com/webstore/detail/vivid-tab/hchlkclbagoklpnijoadpghhcjpeoeim)

### Manual Installation

1. Download the source code or clone the repository:
   ```sh
   git clone https://github.com/jrtilak/vivid-tab.git
   ```
2. Install dependencies:
   ```sh
   pnpm install
   ```
3. Start development mode:
   ```sh
   pnpm dev
   ```
   or build the project:
   ```sh
   pnpm build
   ```
4. Navigate to `chrome://extensions/` in your browser.
5. Enable **Developer Mode** (toggle in the top right corner).
6. Click **Load Unpacked** and select the `build` folder.
